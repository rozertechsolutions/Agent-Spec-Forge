# Structured review

1. Establish scope and expected behavior.
2. Inspect the relevant source material.
3. Verify correctness and completeness.
4. Check security and compatibility where relevant.
5. Separate confirmed findings from uncertainty.
6. Report prioritized findings and minimal corrective actions.
