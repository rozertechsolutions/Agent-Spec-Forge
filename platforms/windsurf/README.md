# Windsurf

Surface:
- IDE

No standalone agent CLI is represented in this package.

Native mechanisms demonstrated:
- Workspace Rules: `.windsurf/rules/*.md`
- Workspace Skills: `.windsurf/skills/<skill>/SKILL.md`
- Workspace Workflows: `.windsurf/workflows/*.md`

Hooks, MCP and other optional customizations are supported but are not required by this minimal
example.

Official documentation:
- https://docs.windsurf.com/windsurf/cascade/memories
- https://docs.windsurf.com/windsurf/cascade/skills
- https://docs.windsurf.com/windsurf/cascade/workflows
