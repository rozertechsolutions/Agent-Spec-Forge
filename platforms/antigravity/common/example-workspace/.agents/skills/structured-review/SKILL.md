# Structured review

1. Establish the exact scope.
2. Inspect the relevant source material.
3. Separate verified findings from assumptions.
4. Prioritize correctness, security, compatibility, and material risks.
5. Report concrete findings with evidence and a minimal corrective action.
6. State explicitly what could not be verified.
