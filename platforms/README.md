# Platform implementation examples

Generic, platform-native examples for AI-Specialized-Departments.

These examples do not contain departments or subdepartments. They demonstrate how a user can
materialize a specialization using each platform's native configuration mechanisms.

Principles:
- `common/`: configuration genuinely shared by independent surfaces of the same product.
- `IDE/`, `CLI/`, `Desktop/`: only surface-specific configuration or usage notes.
- No credentials, tokens, API keys, private endpoints, or model-specific requirements are included.
- The user must adapt the examples to the final specialization.
- Files and paths are based on official documentation reviewed on 2026-07-24.

Included platforms:
1. Antigravity
2. Claude Code
3. Codex
4. Cursor
5. GitHub Copilot
6. Hermes
7. Kiro
8. Windsurf
