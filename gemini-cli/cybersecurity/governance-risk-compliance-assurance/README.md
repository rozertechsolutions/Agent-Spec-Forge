# Gemini CLI Cybersecurity Governance, Risk, Compliance & Assurance

This package uses Gemini CLI native static configuration for a Cybersecurity GRC & Assurance specialization.

## Components

- `GEMINI.md`: persistent area instructions.
- `.gemini/agents/*.md`: local specialist subagents.
- `.gemini/skills/*/SKILL.md`: workspace Agent Skills for repeatable GRC workflows.
- `reference/NATIVE_SOURCES.md`: official documentation basis and native/omitted decisions.

Hooks, MCP servers, settings, commands, extensions, scripts, authentication, and live integrations are intentionally omitted.
