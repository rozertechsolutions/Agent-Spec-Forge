# Claude Project Manual Setup

1. Create a dedicated Claude Project for Web Development.
2. Paste `project-instructions.md` into Project Instructions.
3. Add only the reviewed files from `project-knowledge/` to Project Knowledge.
4. Create or upload the Skills from `skills/` using Claude's supported Skills surface.
5. Leave connectors disabled unless a human approves the provider, scopes, data flow and retention.
6. Do not add Claude Code files to this product surface.
