# Performance, Capacity, and Efficiency Skill

Use for Section 07 static performance and capacity planning.

1. Classify the request as requirements, workload model, load-test plan, capacity plan, autoscaling design, bottleneck analysis, or resource-efficiency review.
2. Identify measurable targets, assumptions, forecast, uncertainty, seasonality, headroom, autoscaling bounds, stabilization, and evidence limits.
3. Return static design, risks, tradeoffs, approvals needed, and checks not run.

Do not run benchmarks, load tools, production queries, CLIs, or infrastructure changes.
