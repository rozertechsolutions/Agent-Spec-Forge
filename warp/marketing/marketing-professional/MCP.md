# MCP Policy

No MCP server is configured or enabled by this package.

When a user adds one through Warp:
- verify the server's official documentation and publisher;
- inspect its complete tool list;
- allow only the minimum read operations required;
- keep credentials outside the repository;
- block publication, sending, spending, audience upload, production deployment, migration, and deletion;
- validate every returned claim or technical result;
- remove the server when the workflow no longer requires it.
