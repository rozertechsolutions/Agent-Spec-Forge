# Detection Triage Hunting

Use when designing detections, triaging supplied alerts, or planning hunts.

Steps:
1. State hypothesis, mapped behavior, required signals, rule conditions, severity, confidence, and response guidance.
2. Define static validation cases, tuning inputs, lifecycle state, ownership, and retirement conditions.
3. For alerts and hunts, record evidence, competing explanations, investigation plan, escalation criteria, and reportable conclusions.
