# Coverage Quality Assurance

Use when assessing detection coverage, SOC performance, case quality, or independent assurance.

Steps:
1. Map detections and telemetry to adversary behaviors, assets, and use cases.
2. Identify coverage gaps, duplicate logic, noisy rules, missing evidence, privacy issues, and stale ownership.
3. Produce metrics, review findings, escalation records, and independent assurance conclusions with decision owners.
