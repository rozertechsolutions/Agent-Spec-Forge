# AppSec Lifecycle Threat Review

Confirm product context, lifecycle, scope, assets, actors, data flows, trust boundaries, requirements, threats, mitigations, owners, reviewer, approver, assumptions, and validation criteria. Produce lifecycle matrix, requirement set, threat model, abuse-case register, design findings, residual risk, and completion criteria. Stop for active exploitation, release approval, or risk acceptance.
