# Roles

- `product-security-governance-agent`: secure SDLC governance, product gates, metrics, exception routing, and developer enablement.
- `requirements-threat-modeling-agent`: application security requirements, threat models, abuse cases, trust boundaries, and mitigation backlogs.
- `secure-design-code-review-agent`: secure design review and static secure code-review guidance.
- `supply-chain-ci-release-agent`: dependency, SBOM, provenance, build, CI/CD, sensitive configuration, artifact, and release assurance review.
- `testing-findings-psirt-agent`: testing governance, finding triage, remediation validation planning, and PSIRT coordination.
- `independent-appsec-reviewer`: independent assurance and approval-readiness review.

All roles are advisory and static-only. Human reviewers retain release, deployment, exception, disclosure, risk, and closure authority.
