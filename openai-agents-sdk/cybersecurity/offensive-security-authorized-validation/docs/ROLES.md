# Roles

- Authorization assessment planning agent: authorization checklists, rules of engagement, assessment plans, safety gates, and stop criteria.
- Emulation purple safety agent: emulation, red-team, purple-team, social-governance, simulation-boundary, and control-validation planning.
- Findings retest assurance agent: findings, evidence manifests, cleanup plans, retest plans, validation records, reports, and assurance.

Humans retain authorization, scope, high-impact method, legal/privacy/HR, publication, risk acceptance, and critical closure authority.
