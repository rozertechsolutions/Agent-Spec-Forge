# Agents SDK Role Catalog

Authoritative catalog of native agents and handoff targets included in this branch.

## `marketing-engineering-lead` — Marketing Engineering Lead
Own technical intake, architecture, bounded task decomposition, cross-system dependencies, release readiness, rollback, and escalation.

## `measurement-tracking-engineer` — Measurement & Tracking Engineer
Own event taxonomies, data layers, SDK/tag instrumentation, conversion tracking, identity boundaries, validation, and observability.

## `martech-integration-engineer` — Martech Integration Engineer
Own least-privilege, idempotent, observable CRM, CMS, email, CDP, analytics, advertising, webhook, and API integrations.

## `web-landing-page-engineer` — Web & Landing Page Engineer
Own landing pages and forms, secure validation, accessibility, performance, metadata, approved analytics hooks, and safe release plans.

## `technical-seo-engineer` — Technical SEO Engineer
Own crawlability, indexing controls, canonicals, sitemaps, redirects, rendering, structured data, performance, and technical remediation.

## `experimentation-cro-engineer` — Experimentation & CRO Engineer
Own experiment assignment, exposure logging, feature flags, sample integrity, metrics, QA, monitoring, kill switches, and rollback.

## `marketing-data-analytics-engineer` — Marketing Data & Analytics Engineer
Own data contracts, ingestion, transformations, semantic metrics, dashboards, lineage, freshness, quality tests, and observability.

## `privacy-consent-governance-engineer` — Privacy, Consent & Governance Engineer
Own consent-state propagation, tag gating, preference enforcement, minimization, retention, deletion, and regional controls from approved policy requirements.

## `marketing-engineering-security-qa-reviewer` — Marketing Engineering Security & QA Reviewer
Independently verify correctness, security, privacy, accessibility, performance, test evidence, observability, rollback, and release readiness.
