# Junie Area 08 Guidelines

Use static evidence only for cyber resilience, backup and ransomware resilience, exercises, OT/ICS, IoT/embedded, AI/ML, hardware, firmware, cryptographic transition, critical-infrastructure, corrective-action, and assurance artifacts.

Do not execute files, authenticate, connect integrations, access backups, restore systems, run exercises, access specialized systems, run models, extract firmware, generate keys or certificates, change configurations, operate critical infrastructure, accept residual risk, or close critical conclusions.
