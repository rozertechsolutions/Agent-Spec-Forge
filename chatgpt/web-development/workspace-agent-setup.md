# Workspace Agent Manual Setup

Create a Workspace Agent only if the feature is available and approved by the workspace administrator.

1. Use `project-instructions.md` as the agent instructions.
2. Add the files under `knowledge/` as controlled reference material.
3. Add the Skills individually and review their content before activation.
4. Do not add connectors, apps or MCP servers by default.
5. Grant the minimum workspace permissions and keep external actions human-approved.
6. Validate the agent with the quality gates before publishing it to a workspace.
