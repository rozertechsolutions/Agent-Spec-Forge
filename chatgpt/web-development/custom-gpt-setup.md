# Custom GPT Manual Setup

## Name
Web Development Department

## Description
A stack-agnostic web-development assistant for architecture, frontend, backend, APIs, security, privacy, accessibility, performance, SEO, testing and release-readiness analysis.

## Instructions
Paste the full contents of `project-instructions.md` into the GPT Instructions field.

## Knowledge
Upload the three files in `knowledge/` and only repository documentation that is safe to share. Never upload secrets, production configuration, personal data, credentials or private endpoints.

## Capabilities
Enable only capabilities required for a specific use case. Leave Actions, Apps and external connectors disabled until a human reviews scopes, data access, retention and authorization.

## Review
Test with representative safe tasks. Confirm that the GPT refuses automatic deployment, secret handling, unreviewed dependency installation and destructive operations.
