# ChatGPT Mobile Development Skills

These are instruction-only Agent Skills for eligible ChatGPT accounts with Skills enabled. Cloning this repository does not install or activate them.

Package one Skill at a time:

1. Review the target `skills/<name>/SKILL.md`.
2. Create a ZIP whose top-level entry is exactly that one Skill directory, containing `SKILL.md`.
3. In ChatGPT, open Skills, choose Create, then Upload from your computer.
4. Review ChatGPT's scan result before installing or sharing.

Do not zip the whole repository, include secrets, add executable helpers, connect apps, configure MCP, run builds, sign artifacts, publish, upload, submit, deploy, or distribute anything from these Skills.
