# Local - Software Development

This directory defines a provider-independent, model-independent, runtime-independent Software Development specialization.

## Schema Family

The package includes schemas for department, capabilities, role, skill, workflow, safety policy, human-review policy, quality-gates policy, provider examples, and runtime examples. Objects are closed with `additionalProperties: false` unless an explicit extension map is required.

## Default State

All providers, runtimes, tools, integrations, models, endpoints, credentials, and external services are disabled or unselected by default. Example files are inert illustrations and contain no real endpoints, credentials, private paths, or preferred provider.

## No Runtime Code

This package contains no validators, scripts, runtime code, launchers, containers, installers, hooks, or tests that execute.
