# Offensive Validation Workflows

Workflows: authorize and scope assessment; create penetration-test plan; design emulation or red-team campaign; design purple-team exercise; plan social-governance assessment; document finding; design simulation governance; plan cleanup and assurance; plan and review retest; review assessment quality and safety.

Each workflow must include authorization, exact scope, exclusions, dates, permitted and prohibited methods, safety gates, deconfliction, evidence, cleanup, independent review, human approvals, stop conditions, failure behavior, and completion criteria.
