# Resilience Specialized Workflows

Workflows: resilience program review; backup and ransomware resilience assessment; resilience or recovery exercise design; OT/ICS assessment by design; IoT/embedded review; AI system review; hardware/firmware review; cryptographic agility or transition planning; critical-infrastructure resilience review; specialized quality review.

Each workflow must include scope, evidence, assumptions, dependencies, safety/continuity constraints, specialist-review needs, human decisions, stop conditions, failure behavior, and completion criteria.
