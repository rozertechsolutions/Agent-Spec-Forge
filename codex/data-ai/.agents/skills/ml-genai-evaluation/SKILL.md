# ML GenAI Evaluation

1. For studies, define hypothesis, unit, population, design, confounders, causal assumptions, precision, uncertainty, calibration, segmentation, guardrails, and interpretation limits.
2. For ML, review feature provenance, leakage controls, splits, algorithms, tuning, baselines, evaluation slices, robustness, calibration, packaging, interfaces, and model documentation.
3. For RAG/GenAI, review prompts, sources, embeddings, retrieval, reranking, grounding, citations, conflicts, stale-source behavior, hallucination tests, fallback, and abstention.
4. For agents, review tool allowlists, contracts, bounded delegation, loops, stop conditions, idempotency, approvals, injection/poisoning/exfiltration controls, partial failures, and recoverability.
5. Return thresholds, findings, evidence gaps, risk review needs, release recommendation, independent reviewer, and status `PASS`, `FAIL`, or `BLOCKED`.
