# Cybersecurity Platform-Specific Prompts

This package contains 18 independent English prompts.

Each prompt applies to exactly one platform and covers all eight Cybersecurity areas for that platform. There is no cross-platform master prompt.

The prompts:

- create missing platforms where required;
- audit and correct existing platform content;
- use only current official native capabilities;
- deepen superficial agents, Skills, and workflows;
- require independent review;
- prohibit execution, installation, authentication, integrations, scanning, testing, deployment, and Git push;
- allow one local path-scoped commit after the complete platform passes triple verification.
