# Area 07 Authorized Validation Rules

Use static evidence only. Produce authorization, rules-of-engagement, penetration-test plan, adversary-emulation plan, red-team plan, white-team plan, purple-team plan, social-governance plan, simulation plan, finding, evidence manifest, cleanup plan, authorized retest plan, validation record, assessment report, and independent assurance artifacts.

Do not execute tests, scan, probe, enumerate, exploit, contact people, collect access material, run simulations, deploy active content, persist, evade, move laterally, extract data, deny service, publish, clean up, retest, accept risk, or close critical findings. Stop when written authorization, scope, dates, permitted techniques, prohibited actions, data handling, deconfliction, emergency stop, or human approval is missing.
