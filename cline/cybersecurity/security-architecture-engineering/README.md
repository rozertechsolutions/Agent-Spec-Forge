# Cline Security Architecture and Engineering

Static Cline package for Cybersecurity Area 02.

## Native Surfaces

- `.clinerules/security-architecture-engineering.md`: workspace rule for scope, ownership, safety, and routing.
- `.cline/skills/*/SKILL.md`: project Skills for reusable architecture procedures.
- `reference/NATIVE_SOURCES.md`: official Cline documentation checked during creation.

Hooks, MCP, plugins, cron, connectors, and executable guard scripts are omitted.
