# Kiro Cybersecurity Application Product DevSecOps Security Specialization

This directory contains the Kiro package for Cybersecurity Area 03: Application, Product, and DevSecOps Security.

Use the steering file in `.kiro/steering/`, the custom agents in `.kiro/agents/`, and the Skills in `.kiro/skills/`. Keep all work static and advisory. Do not execute applications, builds, tests, scanners, package managers, pipelines, deployment commands, or authenticated integrations.

Humans retain final authority for release, deployment, exception approval, disclosure, risk acceptance, and finding closure.
