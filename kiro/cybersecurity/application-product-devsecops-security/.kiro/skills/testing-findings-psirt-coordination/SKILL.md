# Testing Findings PSIRT Coordination

Confirm testing scope, allowed methods, severity model, affected assets, owners, and disclosure boundaries. Plan evidence requirements, triage findings, define remediation validation criteria, draft PSIRT intake and disclosure decision records, and list human approvals. Do not test actively, contact external parties, disclose, close findings, or command incidents.
