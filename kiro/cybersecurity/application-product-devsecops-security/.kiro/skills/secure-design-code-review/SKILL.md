# Secure Design Code Review

Review supplied design and code statically for authentication, authorization, session handling, tenant isolation, validation, output handling, data protection, logging safety, error handling, and secure defaults. Cite evidence, mark inference, assign severity and confidence, list affected assets, actions, validation criteria, limitations, residual risk, and human decisions. Stop before execution or release approval.
